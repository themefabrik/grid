The Grid

12 column based grid

The grid is based on 12 columns which can be combined with another. Gutters between the columns can optionally be turned off with just one .no-gutter class. The max width can be changed by modifying max-width property of the entire .container class wrapped around all grids. Inluding columns will resize accordingly.

Rearrange columns

Columns can be rearranged with the classes .pull-left and .pull-right This is recommendend for two column grids only.
Keep in mind that columns appear on mobile in the same order as they are within the markup.

4
8
Example

<div class="grid">
<div class="col span-4 pull-right">4</div>
<div class="col span-8 pull-left">8</div>
</div>
Image Grids

Create instant grids from one to six columns by wrapping a .content-grid-x class around grid items. This can be useful for an image gallery. This only makes sense if the thumbails have equal dimensions.

Example

<div class="content-grid-4">
<div class="grid-item">
    <img src="img/image.jpg" alt="">
</div>
<div class="grid-item">
    <img src="img/image.jpg" alt="">
</div>
</div>
